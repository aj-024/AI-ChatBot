## AI-ChatBot


This project is a simple AI chatbot that interacts with users through a web interface. The chatbot provides responses based on predefined rules and the capabilities of the Gemini API, which it uses to fetch data dynamically. The project consists of three primary files: index.html, script.js, and style.css.

# Project Structure
index.html: The main HTML file that provides the structure of the chatbot’s web interface.
script.js: JavaScript file that handles the chatbot's interactivity, including sending user queries to the Gemini API and displaying responses.
style.css: CSS file that styles the chatbot interface for a user-friendly experience.

# Features

User-friendly Interface: Simple design for intuitive interaction.
Dynamic Responses: Leverages the Gemini API to provide real-time responses.
Customizable Design: Easily modifiable styles in style.css for a personalized look.

# Requirements

A modern web browser (Chrome, Firefox, Edge, Safari) to open the HTML file.
An internet connection to access the Gemini API for dynamic responses.

# How to Open the Files and Run the Project:

1. Download or Clone the Repository

If you haven’t cloned the repository yet, you can do so by running:

Copy code
git clone https://github.com/your-username/AI-Chatbot-Project.git
Alternatively, you can download the project as a ZIP file and extract it on your computer.

2. Open the HTML File

Navigate to the project folder.
Open index.html with a web browser. This file loads the entire chatbot interface and connects the JavaScript and CSS files.

3. Run the Chatbot

The chatbot will load directly in your web browser.
Start typing a message in the input box, and the chatbot will respond using predefined responses and information fetched from the Gemini API.

# Using the Gemini API

The script.js file contains the logic to interact with the Gemini API.
The chatbot sends user queries to the API, and based on the response from the Gemini API, it displays relevant information to the user.
Note: Make sure you have a valid API key for the Gemini API and have updated the API endpoint and key in script.js (if required).
